# osu!megamix

**Play all osu! modes in one game!**  
Includes **osu!**, **Taiko**, **Catch the Beat (CTB)**, **Mania**, and **Megamix mode**.

---

## ⚡ One-Line Install & Run

Copy-paste this in your terminal to get started immediately:

```bash
git clone https://github.com/zonagreentea/osu_megamix.git && cd osu_megamix && python3 -m pip install --user -r requirements.txt && chmod +x run_osumix.sh && ./run_osumix.sh
